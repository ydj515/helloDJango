# helloDJango

### 실행방법
**python manage.py runserver**


### 명령어
* (models.py 에서) 모델을 변경해 줄 수 있음.
* **python manage.py makemigrations**을 통해 이 변경사항에 대한 마이그레이션을 만들기
* **python manage.py migrate** 명령을 통해 변경사항을 데이터베이스에 적용
* **python manage.py createsuperuser**
